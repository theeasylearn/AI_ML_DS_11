how to fetch data from multiple tables. 
---------------------------------------
when we fetch data from multiple tables, then it is called join operations between tables. first of all of there are many type of join we can use between tables

1) equi/Inner join 
    when we want to fetch data from multiple tables based upon common fields value between tables it is called equi join. here equi join fetch only matching row from both tables means rows which has same value in common fields  

2) Left join 
   we want to fetch data from two tables, we want to fetch all rows from left side table where we want to fetch only matching rows from right side tables based upon common field 

3) Right join 
   we want to fetch data from two tables, we want to fetch all rows from right side table and we want to fetch only matching rows from left side tables based upon common field 

4) full join 
    we want to fetch all rows from two tables without giving any condition on common fields at that time we use full join. it do cartisan product operation on both tables.

Examples
------------------------------------------
fetch all customerNumber,customerName and paymentDate, amount & show them amount wise in descending order 

1st figure which fields belongs to which tables 
    customerNumber,customerName is in customers table 
    paymentDate, amount is in payment table 

2nd fetch these columns from corresponding tables 
    select customerNumber,customerName from customers
    select paymentDate, amount from payments

3rd join them 
    select c.customerNumber,customerName,paymentDate, amount from customers c, payments p where c.customerNumber = p.customerNumber order by amount desc limit 0,1000

fetch all orders number, orderDate, status, productCode, price and quantity and display them orderdate wise 

1st figure which fields belongs to which tables 
    orderNumber,orderDate, status, is in orders table 
    productCode, price and quantity is in orderdetails table 

2nd fetch these columns from corresponding tables 
    select orderNumber,orderDate, status from orders
    select productCode, priceEach , quantityOrdered from orderdetails

3rd join them 
     select o.orderNumber,orderDate, status, productCode, priceEach , quantityOrdered  from orders o, orderdetails d where o.orderNumber=d.orderNumber order by orderdate 


fetch customerNumber,customerName, orderdate, orderNumber, productCode, price, quantity of orders and display them customerNumber wise 

1st figure which fields belongs to which tables 
    customerNumber,customerName is in customers table 
    orderNumber,orderDate, is in orders table 
    productCode, price and quantity is in orderdetails table 

2nd fetch these columns from corresponding tables 
    select customerNumber,customerName from customers 
    select  orderNumber,orderDate from orders 
    select productCode, priceEach, quantityOrdered from orderdetails

3rd join 
    select c.customerNumber,customerName,o.orderNumber,orderDate,productCode, priceEach, quantityOrdered  from customers c, orders o,  orderdetails d where c.customerNumber = o.customerNumber and o.orderNumber = d.orderNumber order by c.customerNumber


fetch productCode, ProductName, quantityOrdered and priceEach and orderNumber and display them orderNumber wise 

1st figure out which fields belongs to which tables 
    productCode, ProductName is in products 
    quantityOrdered and priceEach and orderNumber from orderdetails 

2nd fetch these columns from corresponding tables 
    select productCode,ProductName from products 
    select orderNumber,quantityOrdered,priceEach from orderdetails

3rd join
    select p.productCode,ProductName,orderNumber,quantityOrdered,priceEach from products p, orderdetails od where p.productCode = od.productCode order by orderNumber


Display customerNumber,customerName, and orderNumber and each order total order amount

1st figure out which fields belongs to which tables 
    customerNumber,customerName is in customers
    orderNumber, order total order amount is in orderdetails
    we will fetch one field from order table (connector table )

2nd fetch these columns from corresponding tables 
    select customerNumber,customerName from customers 
    select orderNumber, sum(quantityOrdered * priceEach) from orderdetails group by orderNumber
    select orderNumber from orders 

3d join them 
select c.customerNumber,customerName,o.orderNumber, sum(quantityOrdered * priceEach) as 'total_order_amount' from customers c, orderdetails od, orders o where c.customerNumber = o.customerNumber and o.orderNumber = od.orderNumber group by orderNumber order by c.customerNumber;
