Based on the uploaded SQL notes , here are SQL exercises arranged from beginner to intermediate level.

## Part 1: Table Creation Exercises

1. Create a table named `student` with the following fields:

   * id (Primary Key, Auto Increment)
   * fullname
   * gender
   * dob
   * email (Unique)
   * mobile (Unique)

2. Create a table named `employee` with:

   * empid (Primary Key)
   * empname (Not Null)
   * salary (Check salary > 10000)
   * department
   * joining_date

3. Create a table named `product` with:

   * productid (Primary Key, Auto Increment)
   * productname (Not Null)
   * price (Check price > 0)
   * stock
   * status (Default 'Available')

4. Create a table named `course` with:

   * courseid
   * coursename
   * fees
   * duration

5. Create a table named `department` and `teacher` where teacher references department using a Foreign Key.
